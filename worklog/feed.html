Work Log

今日、何をやれたか。

その日の仕事を、雑なメモで打つだけ。AI が業務日誌に整形して、日次・月次の成果として読み返せます。メモは GitHub の Issue（ラベル worklog）に投げるだけ。スマホの GitHub アプリからでも10秒で書けます。

01 打つ 「+ メモを書く」から Issue にその日の仕事を雑に箇条書き。前日分は本文に「日付: 2026-06-05」と書けばその日に振り分けられます。
02 整う 投稿すると GitHub Actions が起動し、Gemini が見出し・本文・カテゴリ・成果に整形します。
03 まとまる 日次は「今日の総括」、月次は「成果レポート」として自動で積み上がります。
